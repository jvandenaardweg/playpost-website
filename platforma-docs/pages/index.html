Introduction

Platforma 2 Bootstrap is a tool which helps you build responsive HTML prototypes that will work like a finished websites. It contains a vast amount of components which you can use to create your designs. To start working unzip the downloaded package.

Folder Structure

After you download and unzip the package, you’ll find the following folders:

/build – project folder (HTML/CSS), also contains assets with static files (.js libraries and pics);

/html – HTML-version of blocks;

/src – source code;

/src/blocks – blocks’ source code (pug.) divided into categories;

/src/styles – upgraded Bootstrap source styles with pre-configured variables for Platforma.

PUG/SASS Version

Install Node.js, then open directory with the project in the console and install npm packs with the command below:

npm i

Use the command below to compile files from src to build:

gulp build

For working in the Watch mode use the command below:

gulp watch

Layout

There is a file layout.pug in the root directory, which has the basic structure of the template with styles and .js-files. You can also add your own styles and .js-libraries:

<!DOCTYPE html>
html(lang='en')
  head
    block title
      title Platforma 2 Bootstrap - Design Files

    meta(charset='UTF-8')
    meta(name='viewport', content='width=device-width, initial-scale=1.0')
    meta(http-equiv='X-UA-Compatible', content='ie=edge')

    //- CSS import
    link(rel='stylesheet' href='css/main.css')

    //- JS import
    script(src='https://code.jquery.com/jquery-3.3.1.min.js' integrity='sha256-FgpCb/KJQlLNfOu91ta32o/NMZxltwRo8QtmkMRdAu8=' crossorigin='anonymous')
    script(src='https://cdnjs.cloudflare.com/ajax/libs/Swiper/4.4.6/js/swiper.min.js')
    script(src='js/bootstrap.bundle.min.js')

  body
    block content

In order to make a template, create a file in the root directory, for example:

src/my-first-template.pug

You can use this file to modify "content" block from layout.pug. For example, if you want create a file with menu-1, center-5 and footers-2, the final version of your template will look like this:

extend layout

include blocks/menu/menu-1
include blocks/center/center-5
include blocks/footers/footers-2

block content
  .container
    +menu-1()
    +center-5()
    +footers-2()

You'll find the file calledsrc/my-first-template.html with your template in HTML in the /build folder.

SCSS Styles

The workflow is similar to Bootstrap SCSS and all the settings are stored in variables in styles/bootsrap/_variables.scss file. You can customize styles by using those variables, which have intuitively understandable names. We’ve also added a couple of our own additional classes:

.action-link – link with a pointer;

.caret-* – (up, down, left, right) - adds pointer to the block and is used in accordions;

.custom-list – adds bullet styles for numbered and unnumbered lists;

.price-* – (small, medium, large) - used with .price-unit ( e.g.: /mo );

HTML/CSS Version

HTML Structure

/html contains layout.html template, which you can use as a basis of your own custom template. CSS and .js are already embedded:

<!DOCTYPE html>
<html lang="en">

<head>
  <title>Platforma 2 Bootstrap - Design Files</title>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <meta http-equiv="X-UA-Compatible" content="ie=edge" />
  <link rel="stylesheet" href="css/main.css" />
  <script src="https://code.jquery.com/jquery-3.3.1.min.js" integrity="sha256-FgpCb/KJQlLNfOu91ta32o/NMZxltwRo8QtmkMRdAu8=" crossorigin="anonymous"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/Swiper/4.4.6/js/swiper.min.js"></script>
  <script src="js/bootstrap.bundle.min.js"></script>
</head>

<body>
  <!-- your html here -->
</body>
</html>

How to copy HTML-code from layout.html: create my-first-template.html in /build directory, open it in editor and paste the code from layout.html there.

How to make a template from menu-1, center-5 and footers-2: open these blocks from /html directory, copy the HTML-code of an opened block and then paste it into the <body>.

Block are stretched across their entire width. In order to center them use <div class="container">.

Changing Fonts

You can set your own fonts with Google Fonts. Open src/styles/bootstrap.scss and add the font you want:

@import url('https://fonts.googleapis.com/css?family=Some+Font');

Then switch $font-family-base to 'Some Font' in src/styles/_variables file.

If you use CSS version, you’ll also need to add import to css/main.css:

@import url('https://fonts.googleapis.com/css?family=Some+Font');

And then change font-family setting for body tag.

Image & Favicon Settings

We are using placeholder images instead of real images. You can replace placeholder image url with your image url like img/yourimage.jpg

For HTML version:

<img src="image path" alt="image alt text">

For PUG version:

img(src="image path" alt="image alt text")

To change the favicon image of your site, you need to load a new ICO image in a site root or to point out a new address of the image.

<!-- favicon -->
<link rel="shortcut icon" href="images/favicon.png">
<link rel="apple-touch-icon" href="images/apple-touch-icon-57x57.png">
<link rel="apple-touch-icon" sizes="72x72" href="images/apple-touch-icon-72x72.png">
<link rel="apple-touch-icon" sizes="114x114" href="images/apple-touch-icon-114x114.png">
